# 03 — Knowledge Model

## Camadas de conhecimento

O Frankbot organiza conhecimento em quatro camadas.

## 1. Procedimentos

Como executar tarefas recorrentes.

Exemplos:

- ler documentos;
- montar relatorios;
- formular recomendacoes;
- revisar linguagem institucional.

Esse conhecimento deve ficar nas Agent Skills.

## 2. Documentos de referencia

Materiais usados como base.

Exemplos:

- modelos anteriores;
- checklists internos;
- normas;
- contratos;
- pareceres;
- notas tecnicas.

## 3. Estilo de escrita

Padrao textual esperado.

Exemplos:

- tom cauteloso;
- linguagem institucional;
- frases preferenciais;
- forma de conclusao;
- forma de recomendacao.

## 4. Contexto institucional

Informacoes sobre a atuacao da Controladoria.

Exemplos:

- prevencao;
- correcao;
- apoio a secretarias;
- relacao com controle externo;
- logica de recomendacoes;
- diferencas entre controle interno, auditoria e corregedoria.

## Regra

Quando uma tarefa revelar conhecimento de trabalho novo, esse conhecimento deve ser avaliado para virar skill, template, checklist, exemplo ou script.
