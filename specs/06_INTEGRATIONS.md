# 06 — Integrations

## Versao inicial

Na versao inicial, o Frankbot trabalha com textos e arquivos fornecidos pelo usuario.

Nao ha integracao automatica com sistemas internos.

## Integracoes futuras possiveis

### Repositorio de documentos

Possivel integracao com pastas internas, Drive, SharePoint ou sistema de processos.

### Base de relatorios anteriores

Possivel base pesquisavel com relatorios ja produzidos pela Controladoria.

### Banco de dados municipal

Possivel conexao futura com bases estruturadas para analise de dados.

### Editor de documentos

Possivel geracao de saida em Markdown, DOCX ou PDF.

## Regra

Toda nova integracao deve ser registrada antes em:

- specs/05_ACTION_ROUTING_MAP.md
- specs/09_DECISIONS.md
- specs/07_ACCEPTANCE.md
