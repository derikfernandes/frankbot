# Catalogo de Skills do Frankbot

Consultar este arquivo antes de cada tarefa.

Skills iniciais:

1. redacao-relatorio-tecnico
Uso: produzir relatorio, nota tecnica ou analise estruturada.
Saida: documento tecnico estruturado.

2. qa-controladoria
Uso: revisar uma entrega final.
Saida: avaliacao tecnica da entrega.

3. skill-creator
Uso: criar nova skill quando existir uma lacuna de capacidade.
Saida: nova skill proposta.

4. skill-reviewer
Uso: revisar skills ao final de cada tarefa.
Saida: melhoria recomendada.

Skills planejadas:

- leitura-processual
- analise-tecnica-controladoria
- linguagem-institucional
- recomendacoes-e-encaminhamentos

Regra obrigatoria:

Toda entrega deve terminar respondendo:

- A skill foi suficiente?
- Nova skill e necessaria?
- Alguma skill existente precisa ser revisada?
- E necessario criar checklist, template, exemplo ou script?
